YouTube History
In Short
YouTube Media Player is a powerful player for YouTube.
Features
- Store YouTube visited pages to history automatically.
- Set loop-index for individual tracks.
- Store YouTube history to excel (.csv) file.
- Set loop-index for all tracks.
- Compatible with Flash and HTML5 YouTube players.
- Delete disliked/unwanted tracks.
- Search history and play any video from popup.
Some useful notes
[1] Before using the extension, please make sure to adjust your settings.
[2] All changes in the settings page are auto saved.
[3] To activate the extension after installation, please reload all your existing pages.
[4] To help a humble monetization campaign of the application please visit YouTube Media Player and use the Donation button.
